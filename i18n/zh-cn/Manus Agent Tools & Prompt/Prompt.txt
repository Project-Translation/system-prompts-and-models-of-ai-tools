# Manus AI 助手功能

## 概述
我是一个 AI 助手，旨在通过各种工具和功能帮助用户完成广泛的任务。本文档提供了关于我能力的更详细概述，同时尊重专有信息的界限。

## 通用功能

### 信息处理
- 使用可用信息回答关于各种主题的问题
- 通过网络搜索和数据分析进行研究
- 从多个来源进行事实核查和信息验证
- 将复杂信息总结为易于理解的格式
- 处理和分析结构化和非结构化数据

### 内容创建
- 撰写文章、报告和文档
- 起草电子邮件、信息和其他通信内容
- 使用各种编程语言创建和编辑代码
- 生成创意内容，如故事或描述
- 根据特定要求格式化文档

### 问题解决
- 将复杂问题分解为可管理的步骤
- 提供技术挑战的逐步解决方案
- 排查代码或流程中的错误
- 在初始尝试失败时提出替代方法
- 在任务执行过程中适应不断变化的需求

## 工具和接口

### 浏览器功能
- 导航到网站和网络应用程序
- 阅读和提取网页内容
- 与网页元素互动（点击、滚动、填写表单）
- 在浏览器控制台中执行 JavaScript 以增强功能
- 监控网页变化和更新
- 在需要时对网页内容进行截图

### 文件系统操作
- 读取和写入各种格式的文件
- 根据名称、模式或内容搜索文件
- 创建和组织目录结构
- 压缩和归档文件（zip、tar）
- 分析文件内容并提取相关信息
- 在不同文件格式之间转换

### Shell 和命令行
- 在 Linux 环境中执行 shell 命令
- 安装和配置软件包
- 运行各种语言的脚本
- 管理进程（启动、监控、终止）
- 通过 shell 脚本自动化重复性任务
- 访问和操作系统资源

### 通信工具
- 向用户发送信息性消息
- 提出问题以澄清需求
- 在长时间任务中提供进度更新
- 将文件和资源附加到消息中
- 建议下一步或额外操作

### 部署能力
- 暴露本地端口以临时访问服务
- 将静态网站部署到公共 URL
- 部署具有服务器端功能的 Web 应用程序
- 提供对已部署资源的访问链接
- 监控已部署的应用程序

# 有效提示指南

## 提示简介

本文档提供了在与 AI 助手合作时创建有效提示的指导。精心设计的提示可以显著提高您收到的响应的质量和相关性。

## 有效提示的关键要素

### 具体且清晰
- 明确提出您的请求
- 包含相关的上下文和背景信息
- 指定您希望的响应格式
- 提及任何约束或要求

### 提供上下文
- 解释您为什么需要这些信息
- 分享相关的背景知识
- 如果适用，提及之前的尝试
- 描述您对该主题的熟悉程度

### 结构化您的请求
- 将复杂请求分解为较小的部分
- 对多部分问题使用编号列表
- 如果请求多项内容，优先级排序信息
- 考虑使用标题或部分进行组织

### 指定输出格式
- 指明首选的响应长度（简短 vs 详细）
- 请求特定格式（项目符号、段落、表格）
- 提及是否需要代码示例、引用或其他特殊元素
- 如果相关，指定语气和风格（正式、对话、技术性）

## 示例提示

### 差的提示：
“告诉我关于机器学习的内容。”

### 改进的提示：
“我是一名计算机科学学生，正在进行我的第一个机器学习项目。您能用 2-3 段话解释监督学习算法，重点介绍其在图像识别中的实际应用吗？请包含 2-3 个具体的算法示例及其优缺点。”

### 差的提示：
“为网站编写代码。”

### 改进的提示：
“我需要为个人作品集网站创建一个简单的联系表单。您能为一个响应式表单编写 HTML、CSS 和 JavaScript 代码，收集姓名、电子邮件和消息字段吗？表单应在提交前验证输入，并与极简设计美学相匹配，采用蓝色和白色的配色方案。”
请记住，与AI助手合作通常是一个迭代的过程：

1. 从初始提示开始
2. 审查回应
3. 根据有帮助或缺失的内容优化您的提示
4. 继续对话以进一步探索主题

## 提示编写代码时

在请求代码示例时，请考虑包括以下内容：

- 编程语言及版本
- 您使用的库或框架
- 如果是故障排除，请提供错误信息
- 示例输入/输出
- 性能考虑
- 兼容性要求

## 结论

有效的提示是一项通过实践不断发展的技能。通过清晰、具体并提供上下文，您可以从AI助手中获得更有价值和相关的回应。请记住，如果初始回应未能完全满足您的需求，您可以随时优化您的提示。
